# TOTALLY（商品ページ模写）

CodeJump初級編の課題として、アパレルECサイトの商品ページを再現実装しました。  
Flexboxによるレイアウト構築と、テーブル要素の扱い、レスポンシブ対応の理解を目的としています。

---

## 使用技術
- HTML5
- CSS3（Flexbox）
- レスポンシブ対応（@media screen and (max-width: 896px)）

---

## 実装内容
- Flexboxによる商品画像と商品情報の横並びレイアウト
- SP表示での縦並び切り替え（flex-direction: column）
- tableタグを用いたサイズ・数量選択UIの実装
- border・paddingを使ったテーブルデザイン調整
- max-widthを使ったコンテンツ幅制御（1000px / 800px）
- ナビゲーションの横スクロール対応（overflow-x: auto）

---

## 学んだこと
- Flexboxによるレイアウト構築の理解が深まった
- PCとスマホでレイアウトを切り替える設計を実践できた
- tableタグはレイアウトではなく「データ構造」として使うべきだと理解した
- フォーム要素（selectなど）はブラウザ依存で見た目が変わることを学んだ
- 数pxのズレは環境差によるものであり、見た目で判断する重要性を理解した
